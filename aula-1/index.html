Offset de colunas

Utilizando classes de offset como: offset-1, offset-2, etc. para criar espaçamento entre as colunas. O offset é útil para criar espaçamento entre as colunas, deslocando-as para a direita. Ele é especialmente útil quando você deseja criar um layout mais espaçado ou quando precisa alinhar as colunas de uma maneira específica dentro do grid do Bootstrap.
